# Computer-vision-final-project

# Классификация морских животных с использованием нейронных сетей

Курсовая представляет собой desktop-приложение для классификации изображений морских животных с использованием современных архитектур нейронных сетей.

## Основной функционал

### Работа с изображениями
- **Загрузка изображений** - поддержка различных графических форматов
- **Предварительный просмотр** - отображение загруженного изображения перед классификацией
- **Автоматическая предобработка** - адаптация изображения под требования моделей

### Классификация и анализ
- **Многоклассовая классификация** - распознавание 24 различных видов морских животных
- **Вероятностные предсказания** - отображение точности классификации для каждого результата
- **Гибкая система моделей** - поддержка различных архитектур нейронных сетей

### Управление моделями
- **Выбор архитектуры** - возможность работы с различными предобученными моделями (EfficientNetB0, MobileNetV2)
- **Управление чекпоинтами** - загрузка весов с разных этапов обучения
- **Динамическая загрузка** - смена моделей без перезапуска приложения

### Аналитическая информация
- **Детали модели** - отображение технических характеристик архитектуры
- **Метаданные обучения** - информация о эпохах, точности и потерях для каждого чекпоинта
- **Сравнение состояний** - возможность оценить эффективность модели на разных этапах обучения

## Техническая реализация

### Архитектуры моделей
- **EfficientNetB0** - современная эффективная архитектура с оптимизированным соотношением точности и вычислительной сложности
- **MobileNetV2** - легковесная архитектура, оптимизированная для мобильных устройств и быстрого вывода
- **Transfer Learning** - использование предобученных на ImageNet весов с дообучением на целевом датасете

### Процесс обучения
- **Data Augmentation** - расширение датасета через аффинные преобразования и цветовые коррекции
- **Adaptive Learning Rate** - динамическая регулировка скорости обучения в процессе тренировки
- **Structured Checkpointing** - сохранение полного состояния модели с метаданными обучения

### Особенности реализации
- **Модульная архитектура** - возможность легкого добавления новых моделей и архитектур
- **Серийализация метаданных** - сохранение информации об обучении в JSON-формате

## Характеристики моделей
- **Размер входного изображения:** 224×224 пикселей
- **Количество классов:** 24 вида морских животных
- **Аугментация данных:** случайные повороты, сдвиги, масштабирование
- **Оптимизатор:** Adam с адаптивным learning rate
- **Функция потерь:** Categorical Crossentropy
